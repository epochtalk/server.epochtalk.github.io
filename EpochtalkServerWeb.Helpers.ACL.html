    Checks if authenticated User is allowed to perform a specific action.

    Compares a mask all of user's roles into a masked permission set and checks if that permission set contains the specified permission_path.

    Raises CustomErrors.InvalidPermission exception if the User does not have the proper permissions.

    get_user_priority(user)

    @spec get_user_priority(Plug.Conn.t() | EpochtalkServer.Models.User.t()) ::
  non_neg_integer()

    Helper which returns the active User's priority.

    Will return priorty of role with highest permissions if the user is authenticated, otherwise anonymous priority is returned if frontend_config.login_required is false otherwise private role priority is returned. If user is banned the Banned role priority is returned.
